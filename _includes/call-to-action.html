My Linux Background

I have been a Linux user for the past 10 years. During this time I have worked on a variety of Linux related projects. I have installed Linux servers, Linux desktops, and Linux embedded devices.

The distributions I have used include, but are not limited to: Debian, Ubuntu, Linux Mint, Fedora, and Arch. For some of these distributions I have also used them in their server version to host web servers, run media servers, and more.

During this time I have become (what some would call) a Linux power user. I have learned much about how Linux works, how to configure system level files, and how to diagnose and repair broken configurations.

My Programming Background

As I started using Linux I learned the importance of programming. I have since then learned how to program in both Bash and Python. These are both very useful languages for Linux.

Bash is very useful for scripting and automation. Turning somewhat tedious tasks into something that can be run with a single command or even run automatically.

Python is more powerful with more features than bash. It can also help quite a bit with tasks in Linux.
